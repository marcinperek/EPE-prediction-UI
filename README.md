# UI for EPE prediction models
[English](#description) | [Polski](#pl)

## Description
This repository contains Streamlit UI for easy interaction with prediction models resulting from our study of prostate cancer patients.

Currently there are 2 models available for use:
- predicting Extraprostatic Extension (EPE)
- predicting Nodal Involvment (N+)
  
This app is hosted on [Streamlit Community Cloud]()

> [!CAUTION]
> The predictions of the model are not medically verified and must be treated only as a reference. 
> 
> Always consult a medical professional for accurate diagnosis and treatment.


## Future models:
- [] predicting Seminal Vesicle Invasion (SVI)
- [] predicting ISUP grade upgrade
- [] possibility of adding information about biopsy cores to the models

## Opis
Repozytorium zawiera aplikację Streamlit do łatwej interakcji z modelami predykcyjnymi powstałymi w wyniku naszego badania pacjentów z rakiem prostaty.

Obecnie dostępne są 2 modele:
- predykcja Naciekania Pozatorebkowego (EPE)
- predykcja Przerzutów do Węzłów Chłonnych (N+)

Aplikacja jest hostowana na [Streamlit Community Cloud]()

> [!CAUTION]
> Predykcje modelu nie są zweryfikowane medycznie i należy je traktować wyłącznie jako odniesienie.
>
> Zawsze konsultuj się z profesjonalistą medycznym w celu uzyskania dokładnej diagnozy i leczenia.
